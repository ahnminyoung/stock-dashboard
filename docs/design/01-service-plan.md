# 주식/시황 대시보드 서비스 기획서

## 1. 프로젝트 개요

### 1.1 프로젝트 명
**Stock Market Dashboard** - 주식/시황 통합 대시보드 웹 서비스

### 1.2 서비스 목적
- 사용자가 웹/앱 진입 즉시 국내외 주식 시황을 **한눈에** 파악할 수 있는 실시간 대시보드 제공
- 포털형 대시보드 구조로 정보 밀도는 높이되, 시각적 계층을 명확히 하여 복잡도를 낮춤
- 단순 숫자 나열이 아닌 **시각화 중심** 설계로 직관적인 시장 흐름 파악 지원

### 1.3 핵심 가치 제안
1. **통합성**: 국내주식 + 해외주식 + 환율 + 선물 + 뉴스를 한 화면에서 제공
2. **시각성**: 히트맵/섹터맵을 통한 테마/업종별 시장 흐름 직관적 시각화
3. **실시간성**: 실시간 시세 반영 및 시장 상태 표시
4. **맞춤성**: 포틀릿 기반 구조로 사용자 맞춤 대시보드 구성 가능

### 1.4 타겟 사용자
- **주 타겟**: 국내외 주식에 동시 관심 있는 개인 투자자 (20~50대)
- **부 타겟**:
  - 출근 전/퇴근 후 시장 점검이 필요한 직장인 투자자
  - 여러 증권사 앱을 번갈아 확인하는 멀티 플랫폼 사용자
  - 빠른 시장 흐름 파악이 필요한 데이 트레이더

## 2. 서비스 핵심 기능

### 2.1 메인 홈 화면 구성 원칙
```
[철학]
첫 화면 = 시장의 현재를 보는 창
복잡함 없이 핵심만, 하지만 깊이 있게
```

**3층 정보 구조**:
1. **1층 (글로벌 요약 바)**: 전체 시장 스냅샷 - 주요 지수 + 환율 + 시장 상태
2. **2층 (국내주식 모아보기)**: 국내 시장 심층 - 지수 + 히트맵 + 뉴스 + 급등주
3. **3층 (해외주식 모아보기)**: 해외 시장 심층 - 지수 + 섹터맵 + 뉴스 + 빅테크

### 2.2 주요 기능 목록

#### A. 글로벌 요약 바 (Global Summary Bar)
**목적**: 3초 안에 전체 시장 분위기 파악

| 항목 | 표시 정보 | 업데이트 주기 |
|------|-----------|---------------|
| 코스피 | 현재값, 전일대비, 등락률, 상태 | 실시간 (장중), 고정 (장외) |
| 코스닥 | 현재값, 전일대비, 등락률, 상태 | 실시간 (장중), 고정 (장외) |
| 나스닥 | 현재값, 전일대비, 등락률, 상태 | 실시간 (미국 장중), 고정 (장외) |
| S&P500 | 현재값, 전일대비, 등락률, 상태 | 실시간 (미국 장중), 고정 (장외) |
| USD/KRW | 현재 환율, 전일대비, 등락률 | 실시간 (외환시장 개장시) |
| 야간선물 | 코스피200 선물 지수 | 실시간 (야간 거래시) |

**시장 상태 표시**:
- 🟢 개장 전 (Pre-market)
- 🔴 장중 (Trading)
- ⚫ 장마감 (Closed)
- 🟡 애프터장 (After-hours)

#### B. 국내주식 모아보기 섹션

**B-1. 국내 지수 요약 카드**
- 코스피, 코스닥, 코스피200, 선물/옵션 요약
- 각 카드: 미니 차트 (sparkline) + 현재값 + 등락률 + 거래대금
- 클릭 시 해당 지수 상세 페이지 이동

**B-2. 국내 실시간 마켓맵 (핵심 기능)**
```
[마켓맵 사양]
- 형태: 히트맵 (Treemap) 구조
- 단위: 테마 또는 업종
- 박스 크기 결정 요소:
  1. 등락률 (40%)
  2. 거래대금 (40%)
  3. 관심도/검색량 (20%)
- 색상:
  - 빨강 계열: 상승 (진할수록 강한 상승)
  - 파랑 계열: 하락 (진할수록 강한 하락)
  - 회색: 보합
```

**박스 내 표시 정보**:
- 테마/업종명
- 등락률 (큰 글씨)
- 대표 종목 1~2개 (작은 글씨)
- 순위 배지 (1위, 2위, 3위)

**필터 옵션**:
- 유입별 (검색량/조회수 급증)
- 등락률순 (상승률 높은 순)
- 종목 수 (구성 종목 많은 순)
- 기간별 누적 등락률

**기간 선택**:
- 당일 (실시간)
- 5일 누적
- 20일 누적 (1개월)
- 60일 누적 (3개월)

**B-3. 국내 급등/거래대금 탭 포틀릿**
탭 구조:
1. 급등률 상위 (Top Gainers)
2. 거래대금 상위 (Most Active)
3. 조회수 급증 (Trending)

각 탭당 10개 종목 표시:
- 종목명, 현재가, 등락률, 거래대금, 미니 차트

**B-4. 국내 뉴스 포틀릿**
- 최신 국내 증시 뉴스 5개
- 제목, 발행 시간, 출처, 관련 종목 태그
- 클릭 시 뉴스 상세 또는 외부 링크

**B-5. 국내 관심종목 포틀릿**
- 사용자가 등록한 국내 관심종목 실시간 요약
- 최대 10개 표시 (더보기 버튼)
- 종목명, 현재가, 등락률, 간단한 차트

#### C. 해외주식 모아보기 섹션

**C-1. 해외 지수 요약 카드**
- 나스닥, S&P500, 다우, 러셀2000
- 각 카드: 미니 차트 + 현재값 + 등락률 + 보조 지표 (VIX, 10년물 금리 등)

**C-2. 해외 실시간 섹터맵 (핵심 기능)**
```
[섹터맵 사양]
- 형태: 히트맵 구조 (국내와 동일)
- 단위: 섹터 (GICS 기준 또는 자체 분류)
- 박스 크기: 시가총액 + 등락률 가중
```

**주요 섹터**:
- 기술주 (Technology): AAPL, MSFT, GOOGL
- 반도체 (Semiconductors): NVDA, AMD, INTC
- AI/클라우드: NVDA, MSFT, AMZN
- 전기차 (EV): TSLA, RIVN, LCID
- 헬스케어 (Healthcare): JNJ, PFE, UNH
- 금융 (Financials): JPM, BAC, GS
- 에너지 (Energy): XOM, CVX
- 소비재 (Consumer): AMZN, WMT, COST

**박스 내 표시 정보**:
- 섹터명
- 등락률
- 대표 종목 2~3개
- 시총 비중 또는 가중치

**C-3. 해외 인기종목/빅테크 탭 포틀릿**
탭 구조:
1. 인기 종목 (Popular)
2. 빅테크 모니터 (Big Tech)
3. 거래량 상위 (Volume Leaders)

**빅테크 고정 표시**:
- AAPL, MSFT, GOOGL, AMZN, META, TSLA, NVDA
- 각 종목: 현재가, 등락률, 시총, 미니 차트

**C-4. 해외 뉴스 포틀릿**
- 최신 미국 증시 뉴스 5개
- 제목, 발행 시간, 출처, 관련 종목/섹터 태그

**C-5. 해외 관심종목 포틀릿**
- 사용자가 등록한 해외 관심종목 실시간 요약
- 최대 10개 표시

#### D. 하단 확장 영역 (선택적)

**D-1. 경제지표 캘린더**
- 금주 주요 경제지표 발표 일정
- 미국 금리 결정, 고용지표, CPI, GDP 등

**D-2. 시장 심리 지표**
- 공포탐욕지수 (Fear & Greed Index)
- 변동성지수 (VIX)
- Put/Call Ratio

**D-3. 원자재 요약**
- 금 (Gold)
- 은 (Silver)
- 유가 (WTI, Brent)
- 미국 10년물 국채 금리

## 4. 핵심 UX 원칙

### 4.1 정보 계층 구조
1. **Primary**: 주요 지수, 히트맵, 관심종목
2. **Secondary**: 뉴스, 급등주, 거래대금 상위
3. **Tertiary**: 경제지표, 심리지표, 원자재

### 4.2 시각적 우선순위
```
크기: 히트맵 > 지수 카드 > 뉴스/관심종목
색상: 상승(빨강) > 하락(파랑) > 보합(회색)
위치: 상단 > 중단 > 하단
```

### 4.3 인터랙션 원칙
- **즉각성**: 모든 클릭은 0.3초 이내 반응
- **피드백**: 호버 시 툴팁 또는 하이라이트
- **맥락 유지**: 새 탭/페이지 이동 시에도 히스토리 유지
- **에러 핸들링**: 데이터 로딩 실패 시 이전 캐시 데이터 표시 + 안내 메시지

### 4.4 색상 시스템
```css
/* 상승 */
--color-up-primary: #E53E3E;    /* 빨강 */

/* 하락 */
```

## 5. 기술 요구사항

### 5.1 성능 목표
- **초기 로딩**: 2초 이내 (First Contentful Paint)
- **실시간 업데이트**: 1초 간격 (WebSocket 또는 Polling)
- **히트맵 렌더링**: 0.5초 이내
- **페이지 전환**: 0.3초 이내

- Safari (최신 2버전)
- Firefox (최신 2버전)
- Edge (최신 2버전)
- Mobile Safari (iOS 14+)
- Chrome Mobile (Android 10+)

### 5.3 접근성 (Accessibility)
- WCAG 2.1 Level AA 준수
- 키보드 네비게이션 지원
- 스크린 리더 호환
- 색맹 모드 지원 (패턴 추가)

### 6.2 수익화 전략 (향후)
- 프리미엄 구독: 고급 지표, 알림 기능, 광고 제거
- 증권사 제휴: 계좌 연동, 주문 기능
- 광고: 네이티브 광고, 배너 (무료 사용자 대상)

### 6.3 차별화 포인트
1. **국내+해외 통합**: 대부분의 서비스는 분리되어 있음
2. **히트맵 중심**: 직관적 시각화로 차별화
3. **포틀릿 커스터마이징**: 사용자 맞춤 대시보드
4. **모바일 최적화**: 반응형이 아닌 모바일 퍼스트 설계
