# 메인 홈 화면 IA (Information Architecture)

## 1. 전체 사이트 구조

```
Stock Market Dashboard
│
├── 홈 (/)
│   ├── 글로벌 요약 바
│   ├── 국내주식 모아보기
│   └── 해외주식 모아보기
│
├── 국내주식 (/domestic)
│   ├── 시장요약 (/domestic/summary)
│   ├── 실시간마켓맵 (/domestic/heatmap)
│   ├── 업종/테마 (/domestic/sectors)
│   │   ├── 업종별 (/domestic/sectors/industries)
│   │   └── 테마별 (/domestic/sectors/themes)
│   ├── 상승/하락 순위 (/domestic/movers)
│   │   ├── 급등주 (/domestic/movers/gainers)
│   │   ├── 급락주 (/domestic/movers/losers)
│   │   └── 거래대금 상위 (/domestic/movers/volume)
│   └── 뉴스 (/domestic/news)
│
├── 해외주식 (/overseas)
│   ├── 시장요약 (/overseas/summary)
│   ├── 실시간섹터맵 (/overseas/heatmap)
│   ├── 섹터/테마 (/overseas/sectors)
│   │   ├── GICS 섹터 (/overseas/sectors/gics)
│   │   └── 테마별 (/overseas/sectors/themes)
│   ├── 인기종목 (/overseas/popular)
│   │   ├── 인기 TOP 100 (/overseas/popular/top)
│   │   ├── 빅테크 (/overseas/popular/bigtech)
│   │   └── 거래량 상위 (/overseas/popular/volume)
│   └── 뉴스 (/overseas/news)
│
├── 관심종목 (/watchlist)
│   ├── 전체 (/watchlist/all)
│   ├── 국내 (/watchlist/domestic)
│   └── 해외 (/watchlist/overseas)
│
├── 경제지표 (/indicators)
│   ├── 캘린더 (/indicators/calendar)
│   ├── 금리/환율 (/indicators/rates)
│   └── 원자재 (/indicators/commodities)
│
├── 검색 (/search)
│   └── 검색 결과 (/search?q={query})
│
└── 마이페이지 (/mypage)
    ├── 프로필 (/mypage/profile)
    ├── 설정 (/mypage/settings)
    │   ├── 대시보드 편집 (/mypage/settings/dashboard)
    │   └── 알림 설정 (/mypage/settings/notifications)
    └── 히스토리 (/mypage/history)
```

## 3. 네비게이션 구조

### 3.1 Primary Navigation (주 메뉴)

```
┌─────────────────────────────────────────────────────────┐
│ [홈] [국내주식] [해외주식] [관심종목] [경제지표]         │
└─────────────────────────────────────────────────────────┘
```

**메뉴별 하위 구조**:

#### 홈 (/)
- 하위 메뉴 없음 (메인 대시보드)

#### 국내주식 (/domestic)
```
국내주식
├── 시장요약
├── 실시간마켓맵 ⭐ (히트맵)
├── 업종/테마
│   ├── 업종별
│   └── 테마별
├── 순위
│   ├── 급등주
│   ├── 급락주
│   └── 거래대금
└── 뉴스
```

#### 해외주식 (/overseas)
```
해외주식
├── 시장요약
├── 실시간섹터맵 ⭐ (히트맵)
├── 섹터/테마
│   ├── GICS 섹터
│   └── 테마별
├── 순위
│   ├── 인기 TOP
│   ├── 빅테크
│   └── 거래량
└── 뉴스
```

#### 관심종목 (/watchlist)
```
관심종목
├── 전체
├── 국내
└── 해외
```

#### 경제지표 (/indicators)
```
경제지표
├── 캘린더
├── 금리/환율
└── 원자재
```

### 3.2 Secondary Navigation (서브 메뉴)

**우측 상단 유틸리티 메뉴**:
```
[검색 🔍] [관심종목 🔖] [알림 🔔] [마이페이지 👤]
```

- **검색**: 통합 검색 (종목, 지수, 뉴스)
- **관심종목**: 관심종목 빠른 보기
- **알림**: 가격 알림, 뉴스 알림
- **마이페이지**: 프로필, 설정, 대시보드 편집

## 4. 사용자 플로우 (User Flow)

### 4.1 첫 방문 사용자 플로우

```
[홈 진입]
    ↓
[글로벌 요약 바 확인] (3초 이내)
    ↓
[국내 or 해외 섹션 스크롤]
    ↓
[히트맵 상호작용] (클릭, 호버)
    ↓
분기 1: [관심 테마 클릭] → [테마 상세]
분기 2: [관심 종목 검색] → [종목 상세]
분기 3: [뉴스 클릭] → [뉴스 상세]
    ↓
[관심종목 등록] (CTA)
    ↓
[재방문 유도]
```

### 4.2 재방문 사용자 플로우

```
[홈 진입]
    ↓
[글로벌 요약 바 확인] (자동)
    ↓
[관심종목 포틀릿 확인] (1초)
    ↓
분기 1: [히트맵에서 급변 테마 확인]
분기 2: [뉴스 체크]
분기 3: [알림 확인]
    ↓
[심화 분석] (필요 시 상세 페이지 이동)
```

### 4.3 모바일 사용자 플로우

```
[홈 진입 (모바일)]
    ↓
[글로벌 요약 바 가로 스와이프]
    ↓
[국내 섹션 세로 스크롤]
    ↓
[히트맵 카드형 또는 가로 스크롤]
    ↓
[해외 섹션 세로 스크롤]
    ↓
[관심종목 탭 또는 하단 메뉴]
```

### 5.1 포틀릿 분류 체계

```

│
├── 1. 지수/시장 포틀릿
│   ├── 글로벌 요약 바 포틀릿
│   ├── 국내 지수 카드 포틀릿
│   └── 해외 지수 카드 포틀릿
│
├── 2. 시각화 포틀릿 ⭐
│   ├── 국내 실시간 마켓맵 포틀릿
│   └── 해외 실시간 섹터맵 포틀릿
│
├── 3. 순위/리스트 포틀릿
│   ├── 국내 급등/거래대금 탭 포틀릿
│   └── 해외 인기/빅테크 탭 포틀릿

│   ├── 국내 뉴스 포틀릿

│

    ├── 경제지표 캘린더 포틀릿
    ├── 시장 심리 지표 포틀릿
    └── 원자재 요약 포틀릿
```

### 5.2 포틀릿 우선순위 매트릭스

| 포틀릿 이름 | 우선순위 | 필수 여부 | 위치 | 크기 |
|------------|----------|-----------|------|------|
| 글로벌 요약 바 | P0 | 필수 | 최상단 | Full Width |
| 국내 실시간 마켓맵 | P0 | 필수 | 국내 섹션 상단 | Large (2/3) |
| 해외 실시간 섹터맵 | P0 | 필수 | 해외 섹션 상단 | Large (2/3) |
| 국내 지수 카드 | P1 | 필수 | 국내 섹션 시작 | Medium (1/4) |
| 해외 지수 카드 | P1 | 필수 | 해외 섹션 시작 | Medium (1/4) |
| 급등/거래대금 탭 | P1 | 필수 | 국내 섹션 우측 | Medium (1/3) |
| 인기/빅테크 탭 | P1 | 필수 | 해외 섹션 우측 | Medium (1/3) |
| 관심종목 (국내) | P2 | 권장 | 국내 섹션 하단 | Small (1/2) |
| 관심종목 (해외) | P2 | 권장 | 해외 섹션 하단 | Small (1/2) |
| 뉴스 (국내) | P2 | 권장 | 국내 섹션 하단 | Small (1/2) |
| 뉴스 (해외) | P2 | 권장 | 해외 섹션 하단 | Small (1/2) |
| 경제지표 캘린더 | P3 | 선택 | 하단 확장 | Small (1/2) |
| 시장 심리 지표 | P3 | 선택 | 하단 확장 | Small (1/2) |
| 원자재 요약 | P3 | 선택 | 하단 확장 | Small (Full) |

**우선순위 정의**:
- **P0**: 반드시 첫 화면에 표시 (MVP 핵심)
- **P1**: 첫 화면 권장 표시 (MVP 포함)
- **P2**: 사용자 선택 표시 가능 (MVP 포함, 토글 가능)
- **P3**: 추후 확장 또는 프리미엄 기능

## 10. IA 설계 원칙 요약

### 10.1 정보 우선순위 원칙
1. **3초 룰**: 글로벌 요약 바는 3초 이내 파악 가능해야 함
2. **F-패턴**: 좌측 상단부터 시작하여 우측, 하단으로 시선 이동
3. **계층 구조**: 요약 → 상세 → 심화 순서로 정보 제공

### 10.2 일관성 원칙
- 국내 섹션과 해외 섹션은 동일한 구조 유지
- 카드, 포틀릿, 버튼 등 UI 요소의 일관된 스타일
- 색상, 아이콘, 타이포그래피 통일

### 10.3 확장성 원칙
- 포틀릿 기반 구조로 추가/제거 용이
- 새로운 시장 (중국, 일본 등) 추가 가능한 구조
- 사용자 커스터마이징 지원

### 10.4 접근성 원칙
- 키보드 네비게이션 지원
- 명확한 Breadcrumb 제공
- 모바일 터치 최적화 (최소 44x44px 터치 영역)
